## Tech/framework used
#### Built with:
  * React.js
  * Node.js
  * Express.js
  * Knex.js
  * PostgreSQL

## API Reference
This app uses the <a href="https://www.clarifai.com/models/face-detection" target="_blank" rel="noopener noreferrer">Clarifai Face Detection Model API</a>.
